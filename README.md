# Click

Click is a real-time messaging application built using Firebase Realtime Database as its backend. It allows users to chat with each other in real-time, send messages, and create chat rooms.

## Features

- Real-time messaging: Chat with your friends and family in real-time.
- User authentication: Securely authenticate users using Firebase Authentication.
- Create chat rooms: Start private or group chat rooms with other users.
- Message history: View and scroll through your chat history.
- User profiles: Customize your profile with a profile picture and username.

## Technologies Used

- Firebase Realtime Database: For real-time message storage and synchronization.
- Firebase Authentication: For user authentication and authorization.
- Android Studio: The app is developed using Android Studio.
- Java: The primary programming language used for app development.
